# Text_Animation


# 🎬 Text Animation using HTML & CSS

## 📖 Description

This project is a simple text animation webpage that displays a flipping word effect using pure CSS.
It cycles through different words like "Work", "Lifestyle", and "Everything" to create a dynamic and engaging sentence.

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations & Keyframes)

---

## ✨ Features

* Animated text flipping effect
* Smooth CSS transitions
* No JavaScript required
* Simple and clean user interface

---

## 📁 Project Structure

index.html → Main structure of the webpage
Text.css → Styling and animation logic
README.md → Project documentation

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Make sure all files are in the same folder
3. Open `index.html` in any web browser
4. Enjoy the animation

---

## ⚙️ How It Works

The animation is created using CSS keyframes.
Words are stacked vertically inside a container, and the `margin-top` property is animated to create a smooth flipping effect.

---

## 🎨 Customization

* Change the words inside the HTML file (`<div class="box">`)
* Modify colors in the CSS file
* Adjust animation speed in:
  animation: show 1.9s linear infinite;
